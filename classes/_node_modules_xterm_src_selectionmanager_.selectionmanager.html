Class SelectionManager

A class that manages the selection of the terminal. With help from SelectionModel, SelectionManager handles with all logic associated with dealing with the selection, including handling mouse interaction, wide characters and fetching the actual text within the selection. Rendering is not handled by the SelectionManager but a 'refresh' event is fired when the selection is ready to be redrawn.

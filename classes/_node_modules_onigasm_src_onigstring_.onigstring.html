Private _utf8Indexes

_utf8Indexes: UintArray | null
  • Defined in node_modules/onigasm/src/OnigString.ts:17

utf-16 to utf-8 mapping table for all uft-8 indexes starting at _mappingTableStartOffset. utf8-index are always starting at 0. null if there are no multibyte characters in the utf8 string and all utf-8 indexes are matching the utf-16 indexes. Example: _mappingTableStartOffset === 10, _utf16OffsetToUtf8 = [0, 3, 6] -> _utf8Indexes[10] = 10, _utf8Indexes[11] = 13

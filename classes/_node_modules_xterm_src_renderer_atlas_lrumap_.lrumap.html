prealloc

  • prealloc(count: number): void
    • Defined in node_modules/xterm/src/renderer/atlas/LRUMap.ts:59

    Preallocate a bunch of linked-list nodes. Allocating these nodes ahead of time means that they're more likely to live next to each other in memory, which seems to improve performance.

    Each empty object only consumes about 60 bytes of memory, so this is pretty cheap, even for large maps.
